# python-haikang-camera-package

将海康威视工业相机 SDK（MvCameraControl）封装为易用的 Python 接口，支持 GigE 和 USB 设备，提供设备枚举、参数控制、图像采集、图像转换等功能。

## 项目特点

- 纯 Python 实现，基于 `ctypes` 调用官方 DLL
- 面向对象设计，简化相机操作流程
- 支持 **GigE** 和 **USB** 设备（推荐使用 GigE）
- 内置设备列表管理、参数读写、图像获取与显示
- 提供三种封装版本（建议使用 **版本 #3**）
- 兼容 Python 3.8+

## 版本说明

| 版本 | 特点 | 适用场景 |
|------|------|----------|
| #1   | 函数式接口，直接调用 SDK | 快速验证、简单脚本 |
| #2   | 类封装，但参数设置需设备开启 | 中等复杂度的应用 |
| #3   | 完善的状态管理、延迟参数设置、图片对象封装 | 生产环境、复杂项目 |

**建议新项目直接使用版本 #3**。

## 依赖环境

- Windows 操作系统（SDK 仅支持 Windows）
- 安装[海康威视工业相机 SDK](https://www.hikrobotics.com/cn/vision/vision-sdk.htm)（内含 `MvCameraControl.dll`）
- Python 3.8+
- 以下 Python 库：
  - `opencv-python`（用于图像处理）
  - `numpy`
  - `termcolor`（可选，用于彩色输出）

## 安装

将本项目克隆或下载至本地，将 `海康摄像头功能包/摄像头功能包 #3` 目录拷贝到你的项目中使用，或将其添加至 Python 路径。

```bash
git clone https://github.com/Zixu-Toby-He/python-haikang-camera-package.git
```

然后在代码中导入：

```python
from 海康摄像头功能包.摄像头功能包 #3 import 海康摄像头接口
```

## 快速入门（版本 #3）

```python
import cv2
from 海康摄像头功能包.摄像头功能包 #3 import 海康摄像头接口

# 1. 枚举设备
设备列表 = 海康摄像头接口.海康设备列表(["GigE设备"])
相机 = 设备列表[0]  # 取第一台相机

# 2. 打开并设置参数（参数可在打开前设置，打开后自动生效）
相机.插值算法类型 = "高画质"   # 或 0,1,2
相机.曝光时间 = 50.0          # 毫秒
相机.增益 = 20.0
相机.图像捕获帧率 = 30.0      # fps

# 3. 开始采集
相机.打开摄像头()
相机.开始抓取()

# 4. 获取一帧图像（返回图片对象）
图片 = 相机.获取当前图像()
bgr_img = 图片.BGR图像        # 转为 OpenCV BGR 格式
cv2.imshow("Camera", bgr_img)
cv2.waitKey(1)

# 5. 停止并释放资源
相机.结束抓取()
相机.关闭摄像头()
# 程序退出前可关闭设备列表
设备列表.关闭列表()
```

## API 简要说明

### 海康设备列表
- `海康设备列表(检索类别)`：枚举指定类型的设备，返回可迭代对象。
- `关闭列表()`：释放所有相机资源并反初始化 SDK。

### 海康GigE工业相机（设备对象）
属性：
- `IP字符串`、`名称`、`型号`、`开启状态`、`抓取状态`
- `曝光时间`（ms，可读写）、`增益`、`图像捕获帧率`
- `插值算法类型`（可读写，支持 `"低画质"/0`, `"中画质"/1`, `"高画质"/2`）

方法：
- `打开摄像头()` / `关闭摄像头()`
- `开始抓取()` / `结束抓取()`
- `获取当前图像()`：返回 `海康相机图片` 对象
- `销毁摄像头()`：释放内部句柄

### 海康相机图片
- `BGR图像`：返回 `numpy.ndarray` (H,W,3) 的 BGR 图像
- `长`、`高`、`帧号`、`时间戳` 等元信息

## 注意事项

1. **DLL 路径**：确保 `MvCameraControl.dll` 在系统 PATH 或当前工作目录下。
2. **管理员权限**：部分网络相机（GigE）可能需要管理员权限。
3. **像素格式**：默认支持 `BayerRG8`，若相机输出其他格式需自行扩展转换逻辑。
4. **错误处理**：所有操作均会抛出异常，建议捕获并处理。
